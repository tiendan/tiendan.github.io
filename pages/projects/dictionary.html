Zazaki Dictionary

Zazaki is an Indo-European language spoken primarily in eastern Turkey (including my hometown Tunceli). Being a minority language, it lacks easy to use & modern online resources such as dictionaries, learning materials, etc.

This dictionary is my first personal project dedicated to modernizing available online resources. In the (not-so-far) future, it will be made available for the use of all minority languages all over the world.

The system uses ElasticSearch to autocomplete search text, provide "did you mean this?" suggestions and search the given word in the dictionary. The modern front end is written in AngularJS using Angular Material framework.

Technologies Used

Until now, I have used the following tools to develop this project:

  • ElasticSearch
  • AngularJS
  • Angular Material
  • Flask microframework
  • OpenShift free hosting
