Broquil

Broquil is a Django-based management system for consumer cooperatives. It was a personal project mainly with the objective of learning new skills (Django, Bootstrap, etc.) and keeping my old skills fresh (HTML, JS, CSS). I developed the system in my free time over a period of several months and it has been in use since February 2015 in the cooperative El Bròquil del Gòtic, Barcelona.

Technologies Used

Some examples from the tools I used in this project are below:

  • Django 1.8
  • OpenShift free hosting
  • Twitter Bootstrap
  • jQuery & jQuery UI
  • fuelUX
  • Hopscotch (JavaScript product tour)
  • Django Rosetta (localization)
  • Django Chroniker (cron jobs)
  • Google API Python client
  • Python xlrd & xlutils packages (handling Excel files)
