Webcam Eye Tracker

For my PhD thesis, I have been working on algorithms to estimate where someone is looking at using only a webcam. This project is based on the open source OpenGazer software.

In our previous work, we proposed a novel method of estimating the gaze, made it more robust and easier to use. Currently we are working on correcting the deviations caused by head movements.

Technologies Used

During the course of the project I used many libraries/tools including the following:

  • OpenCV
  • Matlab
  • Qt
  • CUDA
  • boost C++ libraries
  • Raspberry Pi (for cheap, standalone eye tracker concept)
  • dlib
  • Multivariate Interpolation (MIR) library
  • Fast Artificial Neural Network (FANN) library
